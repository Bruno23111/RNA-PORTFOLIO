# 🧠 RNA Portfolio - Laboratório de Redes Neurais Artificiais

Este repositório contém o portfólio completo de aprendizagem desenvolvido para o laboratório de Redes Neurais Artificiais (RNA). O projeto consiste em um relatório interativo, organizado em um notebook do Google Colab, que documenta a implementação "from scratch" e a análise de diversas arquiteturas de redes neurais[cite: 1].

## 🎯 Objetivos
*   **Domínio Prático:** Consolidação do uso da linguagem Python e da biblioteca NumPy para o desenvolvimento de sistemas inteligentes[cite: 1].
*   **Análise Arquitetural:** Implementação e estudo do comportamento de modelos que variam desde os clássicos lineares até redes convolucionais modernas[cite: 1].
*   **Metodologia Científica:** Organização de um fluxo de trabalho estruturado, com documentação teórica e análise crítica de cada experimento[cite: 1].

## 📚 Roteiro de Módulos
O portfólio está dividido em 11 módulos obrigatórios, cada um contendo fundamentação teórica e implementação prática[cite: 1]:

1.  **Módulo 01: Perceptron Simples** - O modelo básico de neurônio para classificação linear[cite: 1].
2.  **Módulo 02: Adaline** - Refinamento do aprendizado através do erro linear e gradiente descendente[cite: 1].
3.  **Módulo 03: Multi-layer Perceptron (MLP)** - Redes multicamadas com algoritmo Backpropagation para problemas não-lineares[cite: 1].
4.  **Módulo 04: Avaliação de Redes Neurais** - Implementação de métricas de acurácia e tratamento de saídas[cite: 1].
5.  **Módulo 05: Memórias Associativas** - Sistemas de armazenamento baseados em conteúdo e regra de Hebb[cite: 1].
6.  **Módulo 06: Redes de Hopfield** - Memória autoassociativa com atualização assíncrona e convergência de energia[cite: 1].
7.  **Módulo 07: Mapas de Kohonen (SOM)** - Redes competitivas para aprendizado não supervisionado e clustering[cite: 1].
8.  **Módulo 08: Redes Neurais Sem Peso** - O paradigma WiSARD baseado em endereçamento de memória RAM[cite: 1].
9.  **Módulo 09: Redes ART** - Implementação de Fuzzy ART focada na estabilidade e plasticidade do aprendizado[cite: 1].
10. **Módulo 10: Redes BAM** - Memória Associativa Bidirecional para mapeamento heteroassociativo de padrões[cite: 1].
11. **Módulo 11: Redes Convolucionais (CNN)** - Extração de características espaciais através de operações de convolução[cite: 1].

## 🛠️ Tecnologias Utilizadas
*   **Python 3.x**
*   **NumPy:** Essencial para operações de álgebra linear e manipulação de matrizes[cite: 1].
*   **Markdown:** Utilizado para a documentação e relatórios científicos dentro do notebook[cite: 1].
*   **Google Colab:** Ambiente de desenvolvimento e execução em nuvem[cite: 1].

## 📈 Critérios de Integridade
Este projeto foi avaliado com base na integridade da resolução de todos os módulos, no rigor metodológico das explicações teóricas e na profundidade da capacidade analítica sobre os resultados obtidos[cite: 1].
