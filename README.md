# 🧠 RNA Portfolio - Laboratório de Redes Neurais Artificiais

Este repositório contém o portfólio completo de aprendizagem desenvolvido para o laboratório de Redes Neurais Artificiais (RNA). O projeto consiste em um relatório interativo, organizado em um notebook do Google Colab, que documenta a implementação "from scratch" e a análise de diversas arquiteturas de redes neurais.

## 🎯 Objetivos
*   **Domínio Prático:** Consolidação do uso da linguagem Python e da biblioteca NumPy para o desenvolvimento de sistemas inteligentes.
*   **Análise Arquitetural:** Implementação e estudo do comportamento de modelos que variam desde os clássicos lineares até redes convolucionais modernas.
*   **Metodologia Científica:** Organização de um fluxo de trabalho estruturado, com documentação teórica e análise crítica de cada experimento.

## 📚 Roteiro de Módulos
O portfólio está dividido em 11 módulos obrigatórios, cada um contendo fundamentação teórica e implementação prática:

1.  **Módulo 01: Perceptron Simples** - O modelo básico de neurônio para classificação linear.
2.  **Módulo 02: Adaline** - Refinamento do aprendizado através do erro linear e gradiente descendente.
3.  **Módulo 03: Multi-layer Perceptron (MLP)** - Redes multicamadas com algoritmo Backpropagation para problemas não-lineares.
4.  **Módulo 04: Avaliação de Redes Neurais** - Implementação de métricas de acurácia e tratamento de saídas.
5.  **Módulo 05: Memórias Associativas** - Sistemas de armazenamento baseados em conteúdo e regra de Hebb.
6.  **Módulo 06: Redes de Hopfield** - Memória autoassociativa com atualização assíncrona e convergência de energia.
7.  **Módulo 07: Mapas de Kohonen (SOM)** - Redes competitivas para aprendizado não supervisionado e clustering.
8.  **Módulo 08: Redes Neurais Sem Peso** - O paradigma WiSARD baseado em endereçamento de memória RAM.
9.  **Módulo 09: Redes ART** - Implementação de Fuzzy ART focada na estabilidade e plasticidade do aprendizado.
10. **Módulo 10: Redes BAM** - Memória Associativa Bidirecional para mapeamento heteroassociativo de padrões.
11. **Módulo 11: Redes Convolucionais (CNN)** - Extração de características espaciais através de operações de convolução.
